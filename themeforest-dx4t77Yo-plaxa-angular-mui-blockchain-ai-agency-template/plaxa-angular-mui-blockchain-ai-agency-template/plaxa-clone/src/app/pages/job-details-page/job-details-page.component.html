Description & Requirements

Plaxa is a leading cybersecurity services provider dedicated to safeguarding organizations from cyber threats and ensuring the security of their digital assets. We are seeking a highly motivated and skilled Cybersecurity Analyst to join our team in protecting our clients from ever-evolving security risks. Over the past few years, Forward Networks has received tremendous industry recognition, including “Cool Vendor in Enterprise Networking” by Gartner, “Product of the Year” by Cloud Computing, “Enterprise Cloud Computing Software of the Year,” and has been named to Fortune’s 2022 “Best Workplaces in the Bay Area” list.

Forward Networks is looking for a front-end engineer to join our UI team and help us build new features for our flagship web application. In this position you will have the opportunity to use the latest web technologies to build a beautiful, highly-interactive, single-page web application that’s currently used by some of the world’s largest companies to help manage their business-critical network infrastructure.

Requirements:

  • 5+ years of professional front-end engineering experience
  • Ability to write high-quality and unit-tested JavaScript
  • Comfortable debugging and profiling web apps with browser dev tools
  • Care about code quality, user experience, and visual design

Additional Experience:

  • Mastery of CSS and HTML fundamentals
  • Familiarity with the latest CSS layout standards (flexbox, grid)
  • Comfortable working directly with Web APIs and the DOM
  • Experience building single-page applications using at least one of the major front-end frameworks (React, Angular, Vue)
  • A good eye for visual design; you get annoyed when things aren't pixel-perfect
  • Knowledge of patterns and best practices related to interaction design and information architecture

Salary: $180,000-$200,000
